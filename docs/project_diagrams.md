# Диаграммы проекта Library IS

Файл `docs/project_diagrams.html` содержит две готовые диаграммы для документации:

- диаграмма вариантов использования;
- карта функций MVP/Future.

Открой `docs/project_diagrams.html` в браузере и сделай скриншоты нужных блоков для вставки в курсовую.

## Краткое описание диаграммы вариантов использования

В системе выделены три роли:

- `Гость` просматривает книги, выполняет поиск, смотрит филиалы и может зарегистрироваться.
- `Студент` входит в систему, создает заявки на выдачу и возврат книги, просматривает историю выдач.
- `Администратор библиотеки` управляет книгами, филиалами, факультетами, подтверждает заявки студентов и формирует отчеты.

## Краткое описание карты функций

Раздел `MVP` включает реализованные функции проекта: просмотр каталога, роли пользователей, заявки на выдачу и возврат, управление книгами/филиалами/факультетами, отчеты и триггеры БД.

Раздел `Future` содержит функции, которые можно указать как направления развития: QR-коды экземпляров, бронирование, уведомления, экспорт отчетов, аудит действий и расширенный поиск.
