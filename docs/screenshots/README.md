# Скриншоты интерфейса для отчета

Сохраните в эту папку скриншоты с такими именами:

1. `01-dashboard.png` — страница `/dashboard` (статистика + быстрый поиск)
2. `02-books.png` — страница `/books` (таблица + фильтры)
3. `03-book-modal.png` — модальное окно добавления/редактирования книги
4. `04-branches.png` — страница `/branches`
5. `05-branch-details.png` — модальное окно "Посмотреть книги" по филиалу
6. `06-reports.png` — страница `/reports` (форма + результат)
7. `07-top-issued.png` — TOP 10 самых выдаваемых книг
8. `08-branch-delete-error.png` — ошибка при удалении филиала с книгами
9. `09-pgadmin-schema.png` — структура таблиц в pgAdmin

Рекомендуемый формат: PNG, ширина 1366 px (или выше).
